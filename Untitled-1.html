O que é HTML?

Por definição...

HTML significa HyperText Markup Language, que em português é Linguagem de Marcação de Hipertexto É a linguagem base para criar páginas na internet. O HTML serve para estruturar o conteúdo de um site, como textos, imagens, links, tabelas, entre outros. O HTML é basicamente o 'esqueleto'. Os músculos, a pele, o cabelo... tudo que é mais 'estético' é competência de outras linguagens, que veremos posteriormente. HTML dá corpo ao que o CSS embeleza e o JavaScript anima.

Para que serve o HTML

  • Definir títulos
  • Inserir imagens
  • Criar links
  • Organizar parágrafos
  • Montar tabelas e listas
    • *

As principais tags

Tags
H1 até H6 P A Img UL, OL e LI Table, TR e TD Div e Span
Função
Títulos com diferentes níveis de importância Parágrafos Links Imagens Listas (não ordenadas e ordenadas) Tabelas Divisões de conteúdo e estilos
Conclusão

O HTML é fundamental para a construção de sites e páginas na internet. Sem ele, os navegadores não conseguiriam exibir conteúdos de forma organizada. Mesmo com o avanço de outras linguagens (como CSS e JavaScript), o HTML continua sendo a base de tudo.

"HTML é a espinha dorsal da web — sem ele, nada ganha forma. É como construir uma casa: tijolo por tijolo, tag por tag."
